Stop sending notifications to affiliates for referrals that have a subtotal of zero.

== Description ==

Checks the total for an affiliate referral and if the total is zero, no notification will be sent to the affiliate.
It also deleted the referral with a subtotal of zero, so that it does not show up in the affiliate's history.

== Installation ==

1. Upload directory `affiliates-pro-zero-notifications` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
